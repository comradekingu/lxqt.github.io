The LXQt team is proud to announce the release of lxqt-globalkeys 0.16.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-globalkeys/releases).

Main changes:

 * Disabled global shortcuts aren't blocked in apps anymore.
 * Daemon restart isn't needed for enabling a shortcut.

<br/>
A full list of changes is in the CHANGELOG file.
